# Sarah Jing’s Personal Blog

Welcome! This is my personal academic website and blog—a place for research, study notes, projects, photography, travel diaries, and the small things I learn along the way.

I am a PhD student at the NAIST Biomedical Image Intelligence Lab, working on federated learning, medical image analysis, and quantitative musculoskeletal CT analysis. Outside research, I enjoy bouldering, photography, and traveling.

The website is built with simple static HTML, CSS, JavaScript, and Markdown so it can run directly on GitHub Pages and remain easy to update.

New notes and photographs will be added gradually. Thanks for visiting.
